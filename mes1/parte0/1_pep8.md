Conjunto de recomendacao de estilo para escrever python
    - Identacao de 4 espacos
    - linhas com no maximo 79 caracteres.
    - uso de nome descritivos para variaveis,funcoes,classes etc..
    - uso de linhas em brancos para separar blocos de codigo e funcoes.
    - snake_case para funcoes e variaveis, CamelCase para classes.
